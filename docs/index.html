macOS Markdown Reader

OpenNow

OpenNow is a read-only Markdown reader for macOS built around fast launch, local files, clean typography, and a native outline-first reading flow.

Support Privacy Policy GitHub

What OpenNow Does

Opens local .md and .markdown files.

Shows a native outline alongside rendered Markdown content.

Supports headings, lists, tables, code blocks, images, dark mode, and keyboard paging.
