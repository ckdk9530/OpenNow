What To Include In A Support Request

  • Your macOS version.
  • The OpenNow version you are using.
  • A short description of what you expected and what actually happened.
  • A sample Markdown file or screenshot if the issue is about rendering.

Common Topics

Files or images will not open
OpenNow is designed for sandbox-safe local access. If a Markdown file references relative images, select the file or its folder when prompted so OpenNow can keep access to that directory tree.

Rendering looks wrong
Send the Markdown file, a screenshot, and note whether the problem affects tables, code blocks, images, or headings.

App Store purchases
OpenNow does not offer in-app purchases, subscriptions, or paid feature unlocks.
